Ultimate-Guitar UX Redesign

James Yoon's UX Project for Digital Humanities 110 (S'22)

This project involved redesigning Ultimate-Guitar to be more user-friendly for both beginner and advanced guitar players.

1: Heuristic Analysis

I conducted heuristic analysis on the original Ultimate-Guitar site to identify potential design flaws.

Read more about heuristic analysis
2: Usability Testing

I conducted a pilot usability test on the original Ultimate-Guitar site.

Read more about usability testing
3: Contextual Inquiry

Using an interview and ethnography, I studied how musicians in an ensemble interacted with their music.

Learn about my research insights
4: UX Storytelling

I created two personas and two scenarios based on my research.

Explore the personas and scenarios
5: Wireflow and Flow

Using the personas and scenarios, I designed a low-fidelity prototype.

6: Interface Design

I created different interface designs on the homepage.

7: High-Fidelity Prototyping

I translated my low-fidelity prototype into an interactive one and improved it through a cognitive walkthrough and 4 usability tests.
